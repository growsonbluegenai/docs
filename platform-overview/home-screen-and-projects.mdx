When you log in to BlueGenAI, you land on the project dashboard. This is your home base — where you create new projects, open existing ones, manage collaboration, and track your token usage.

---

## How to create an account

1. Go to [BlueGenAI.ai](https://bluegenai.ai)
2. Click **Sign Up**
3. Enter your email address and set a password
4. Verify your email — check your inbox for a verification link
5. Log in — new accounts receive free platform credits automatically, no purchase required

---

## How to navigate the project dashboard

The project dashboard shows all your projects as cards. Each card displays the project name, client, and description.

| Dashboard element | What it does |
| --- | --- |
| **Create a Project** button | Opens the new project dialog |
| Project card | Click to open the project |
| More options menu (⋯) | Access collaboration settings, billing preferences, and project management options |
| Token usage (top-right) | Shows your total usage broken down by Design Mode and Build Mode |

---

## How to create a new project

1. Click **Create a Project** on the project dashboard
2. Enter a project name — there are no restrictions on naming
3. Optionally add a client name and description
4. Click **Create** — your project opens in Design Mode

---

## How to open an existing project

1. Locate the project card on the dashboard
2. Click the card — the project opens in the mode it was last used

---

## How to manage a project from the dashboard

1. Click the **more options menu** (⋯) on the project card
2. From here you can:
   - Invite collaborators and assign roles
   - Configure billing preferences
   - Access project settings
   - Generate shareable links for stakeholders

---

## How to delete a project

Only the project Owner can delete a project.

1. Click the **more options menu** (⋯) on the project card
2. Select **Delete Project**
3. Confirm the deletion

<Warning>
  Deleting a project is permanent. All project files, design artifacts, build history, and collaboration settings are removed. This action cannot be undone.
</Warning>

---

## Understanding project roles

When you create a project you are automatically its Owner. When you invite others they are assigned one of three roles:

| Role | What they can do |
| --- | --- |
| **Owner** | Full control — edit, manage collaboration, configure billing, delete the project |
| **Editor** | Contribute to design and build; invite additional collaborators |
| **Viewer** | Read-only access for review and feedback |

For full details on inviting collaborators and managing access see [Collaboration & Sharing](/platform-overview/collaboration-and-sharing).
